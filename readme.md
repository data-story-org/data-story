See [contributing](contributing.md) for setuping seamless development environment for core and gui.

Build web files

```
yarn && yarn watch
```

Serve public/index.html

## Hotkeys (MAC)

| Hotkey              |      Action      |
| ------------------- | :--------------: |
| ALT+D               |  Go to diagram   |
| ALT+T               | Go to inspectors |
| ALT+PLUS            |     Add node     |
| ALT+R               |   Run diagram    |
| SHIFT+click on link |   Select link    |
